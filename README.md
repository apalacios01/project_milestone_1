# Web Cookies and Privacy Website

## Overview
This project is a multi-page website created for IT 3203. The purpose of this website is to explain how web cookies work, why they are important, and how they relate to user privacy. It focuses on both the technical side of cookies and the real-world concerns around data collection and tracking.

## Project Goals
The goal of this project was to create a multi-page website that explains web cookies, how they work, and why they matter for online privacy. The website is meant to help users understand both the technical side of cookies and the real-world concerns around tracking, consent, and data collection. For Milestone 3, the main objective was to improve the website so it works well on desktop, tablet, and mobile screens while also finalizing the research content.


## Features
- 6-page website
- Navigation bar across all pages
- Structured content using HTML
- Styled layout using CSS
- Images to support content
- Table of key concepts
- References page with sources
- About page with author and project details
- quiz to test yourself 

## Pages Included
- **Home (index.html):** Introduction to cookies and overview of the topic  
- **Cookie Basics (basics.html):** Explains what cookies are and how they work  
- **Privacy Issues (privacy.html):** Discusses concerns related to tracking and data collection  
- **Key Concepts (concepts.html):** Defines important terms related to cookies and privacy  
- **References (references.html):** Lists all sources used for research  
- **About (about.html):** Introduces the author and explains the purpose of the project
- **Quiz (quiz.html):** Take a quiz to test yourself!

## What I Learned
This project helped me better understand how to structure a website using semantic HTML and how to apply CSS to create a consistent layout. It also showed me that building a website takes more time and attention to detail than expected. Small issues like file paths, layout alignment, and navigation consistency required troubleshooting and patience.

In addition to technical skills, I learned more about how cookies work behind the scenes and how they impact user privacy. This topic made me think more about the responsibility developers have when handling user data.

## Challenges
One of the main challenges was making sure all pages were consistent and connected properly. Debugging issues like broken images, incorrect file paths, and styling problems took time. It also took effort to organize the content in a way that was clear and easy to follow.
