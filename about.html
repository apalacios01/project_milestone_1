About this Project

This website was created as part of the IT 3203 Intro to Web Development course. The goal of this project was to design and build a multi-page website using HTML and CSS while presenting research on a web-related topic.

I chose to focus on cookies and privacy because it connects both technical functionality and real-world concerns. While cookies improve user experience by remembering information, they also raise important questions about tracking, transparency, and how data is collected online.

This project helped me improve my understanding of semantic HTML, CSS styling, and how to organize information in a way that is clear and easy for users to navigate. At the same time, it also showed me that building a website is not as simple as it looks. Even a basic site requires attention to detail, consistency across multiple pages, and time spent fixing small issues. Overall, the process was a bit tedious at times, but it helped me better understand what goes into creating a functional and well-structured website.
